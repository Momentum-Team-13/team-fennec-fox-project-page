Project Description

It ain't easy being green. PlantSpace is here to help.

PlantSpace is an online forum for all things green. We designed an application that uses Django and DjangoREST Framework on Heroku, and ReactJS on Netlify. This site allows user to ask and answer questions related to all things plants. Any user may look at questions and answers, but if a user registers and logs in, they may post their own questions and answers. A user may also delete their questions if they want. This websites grants users a space to ask any and all questions about their beloved plants.
